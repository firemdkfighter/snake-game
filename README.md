# Snake Game

Browser-based Snake game built with vanilla JavaScript and Vite.

## Quick start

```bash
docker compose up -d --build
```

Open http://localhost:8080

## Stop

```bash
docker compose down
```

## Controls

- Arrow keys or WASD to change direction
- Touch/swipe on mobile devices
- Click "Play Again" or press Enter/Space after game over

## Development

```bash
# Build and run in Docker (do NOT run directly on host)
docker compose up -d --build

# Rebuild after changes
docker compose up -d --build
```
